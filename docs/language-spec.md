# BadLang Language Specification

BadLang is a statically typed, compiled language with a focus on performance and simplicity.

## Features
- **Static Typing**: Type checking at compile time.
- **Functions**: First-class support for functions.
- **OOP**: Support for classes and objects (in progress).
- **Modules**: Module-based code organization.
- **LLVM Backend**: High-performance compilation.
- **Interpreter**: Fast feedback loop for development.
